ARCH012 ARCH DI ✓ レビュー済

ファクトリ関数 bs.Spot() を唯一の公開エントリーポイントとし、全コンポーネント（DB、ストレージ、シリアライザ、リミッター等）のデフォルト組み立てを行う。

core.Spot.__init__ は全依存を明示的に要求し、暗黙のデフォルトを持たない。全コンポーネントは Protocol / ABC に基づくインターフェースで定義され、テストや拡張時の差し替えを容易にする。

親: REQ012

子: SPEC022

SPEC022 SPEC DI ✓ レビュー済

bs.Spot(name, ...) ファクトリ関数が唯一のパブリックエントリポイント。 引数未指定時のデフォルト: - SQLiteTaskDB (.beautyspot/{name}.db) - LocalStorage (.beautyspot/blobs/{name}) - MsgpackSerializer - TokenBucket (10000 TPM) - WarningOnlyPolicy

全コンポーネントは Protocol/ABC で定義: TaskDBCore, BlobStorageCore, SerializerProtocol, StoragePolicyProtocol, LimiterProtocol。

core.Spot は CacheManager と LimiterProtocol を直接受け取り、 ファクトリ外での直接インスタンス化は非推奨。

親: ARCH012
