SPEC021 SPEC CLI ✓ レビュー済

beautyspot CLIコマンド群を提供する（Typer ベース）。 list: キャッシュ済みタスクまたは利用可能なDBの一覧表示。 show: 特定キャッシュキーの詳細表示。 stats: キャッシュ統計の表示。 clear: キャッシュの削除。 clean: 孤立Blobファイルのガベージコレクション。 gc: 期限切れタスクと孤立ストレージの一括クリーンアップ。 prune: 時間ベースの一括削除。 ui: インタラクティブダッシュボードの起動。 version: バージョン情報の表示。 出力は rich (Console, Table, Panel) でフォーマットする。
