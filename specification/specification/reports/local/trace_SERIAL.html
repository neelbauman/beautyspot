アイテム詳細

REQ005 REQ SERIAL ✓ レビュー済

関数の戻り値を安全にシリアライズ・デシリアライズできること。ユーザー定義型のカスタムエンコーダ・デコーダを登録可能であること。

親: —

子: SPEC012, SPEC013

SPEC012 SPEC SERIAL ✓ レビュー済

MsgpackSerializer は MessagePack ベースのシリアライザ。 dumps(obj) -> bytes と loads(data) -> Any を提供する。 カスタム型は ExtType (code 0-127) で拡張する。 スレッドセーフ: レジストリは Copy-on-Write、サブクラスキャッシュはスレッドローカル LRU。 MRO スキャンによりサブクラスの自動解決を行う。

親: REQ005

子: IMPL012, TST012

SPEC013 SPEC SERIAL ✓ レビュー済

spot.register(code, encoder, decoder) デコレータまたは spot.register_type(cls, code, encoder, decoder) メソッドにより、 カスタム型をシリアライザに登録する。code は 0-127 の一意な整数。 同一コードまたは同一型の重複登録は ValueError で拒否する。 decoder_factory オプションで型に基づくデコーダ自動生成もサポートする。

親: REQ005

子: IMPL013, TST013

IMPL012 IMPL SERIAL ✓ レビュー済

親: SPEC012

IMPL013 IMPL SERIAL ✓ レビュー済

MsgpackSerializer.register() メソッドの実装。 code 0-127 バリデーション、重複チェック、 Copy-on-Write による安全な登録を含む。 Spot.register()/register_type() によるユーザー向けインターフェースも含む。
